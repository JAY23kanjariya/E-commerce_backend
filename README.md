# E-Commerce Backend (Core PHP)

A scalable and professional E-Commerce Backend API built using Core PHP, MySQL, and MVC Architecture.

This project follows clean coding standards, modular architecture, and professional backend development practices similar to modern PHP frameworks.

---

# 🚀 Tech Stack

- PHP 8+
- MySQL
- Composer
- PDO
- MVC Architecture
- REST API Structure
